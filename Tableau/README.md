# Tableau Dashboards & Visualizations

This folder contains Tableau workbooks and visualizations for the Customer Churn Prediction project.

## Contents

- **churn_analysis.twbx** - Main dashboard showing churn analysis and trends
- **customer_insights.twbx** - Customer segmentation and behavior visualizations
- **PDF_exports/** - PDF versions of dashboards for sharing

## Usage

1. Open `.twbx` files with Tableau Desktop or Tableau Public (free version)
2. Connect to your data source as needed
3. Interact with filters and drill-down features

## Notes

- Requires Tableau Desktop or Tableau Public to view/edit workbooks
- PDF exports are provided for quick viewing and sharing
